zyX Library

A lightweight, template-based JavaScript library with reactive variables and easy DOM manipulation

Tagged Templates

Use familiar template literal syntax with the powerful html tag for expressive templating.

Dynamic Variables

Create reactive variables with dynamicVar that automatically update the DOM when their value changes.

Element References

Easily reference elements with the this="name" attribute to access them directly in your code.

Event Binding

Handle events with inline functions using zyx-event attributes for clean, declarative code.

Template Chaining

Chain template methods like bind(), join(), place(), and appendTo() for flexible DOM manipulation.

Dynamic Components

Embed components directly in templates for seamless composition of complex interfaces.
